
# 🐬 SQL Practice Repository

이 레포지토리는 데이터리안(Datarian) 학습 과정 및 개인적인 SQL 역량 강화를 위해 푼 문제들의 정답 코드를 기록하고 복습하기 위한 공간입니다.

## 📌 문제 출처
본 레포지토리에 수록된 모든 문제의 출처는 데이터 분석가를 위한 SQL 연습 플랫폼, **[solvesql(솔브에스큐엘)](https://solvesql.com/)** 입니다.

## 🗂️ 디렉토리 구조 (Directory Structure)
문제들은 학습과 복습이 용이하도록 **난이도별**로 분류되어 있습니다.

```text
📦 solvesql-practice
 ┣ 📂 Level_1
 ┃ ┣ (현재 Level 1의 문제들은 생략했습니다.)
 ┣ 📂 Level_2
 ┃ ┣ 📜 ...
 ┣ 📂 Level_3
 ┃ ┗ 📜 ...
 ┣ 📂 Level_4
 ┃ ┗ 📜 ...
 ┗ 📜 README.md
